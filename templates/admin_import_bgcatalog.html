Import from BG Catalog

Upload a BG Catalog export .zip (the JSON file plus any photos, all in one archive). This is meant to be run once — running it again will re-import everything as duplicates.

  • Games are matched to existing ones by their BoardGameGeek ID, or added if new.
  • Every player becomes a guest, except plays marked as yours in the export, which link to your account.
  • Imported plays default to private visibility — only you can see them until you change that.
